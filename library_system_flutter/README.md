# 図書館オンラインサービス

学校PCでの利用を前提にした `Flutter Web` ベースの図書館オンラインサービスです。

## 現在の状態

- `利用者画面` と `管理者画面` の初期UIを実装
- `モックデータ` でログイン、予約、貸出管理、返却、本登録、申請承認、集計を確認可能
- `クラス表示` を管理者画面で見やすく表示
- 将来的に `Firebase` へ差し替えやすい構成で整理

## 画面

- 利用者
  - ログイン
  - 予約
  - 登録申請
  - 借りている本
  - 履歴

- 管理者
  - ログイン
  - 予約本確認
  - 配達済み管理
  - 本登録・解除
  - 申請承認
  - 集計
